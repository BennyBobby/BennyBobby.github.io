Brain Tumour Classification

This project involved the design and implementation of a custom Convolutional Neural Network (CNN) architecture using the PyTorch framework for multi-class brain tumor classification from MRI images, aiming to accurately distinguish between glioma, meningioma, and pituitary tumors, as well as healthy cases.

Following 15 training epochs, the model achieved great diagnostic performance on the validation set, highlighted by:
Overall Accuracy: 93.55%
F1-Score: 93.46%

The sensitivity across critical categories was particularly high:
Recall for glioma tumors: 97.13%
Recall for pituitary tumors: 94.61%
Precision for no tumor (Reliability of Negative): 93.24%

Skills : Computer Vision, PyTorch, Convolutional Neural Network, Python

Chess Data Analysis

This project is a comprehensive Data Analysis and Visualization initiative focused on extracting insights from a large dataset of player games sourced from Chess.com.

The workflow involved a complete ETL (Extract, Transform, Load) pipeline, beginning with data cleaning and transformation to ensure data quality.
The final insights, including key performance indicators (KPIs) and statistical trends, are presented through a series of graphs and interactive data displays.

The project culminates in a Streamlit application that serves as an interactive dashboard, allowing users to explore the results dynamically.

Skills: Pandas, Data Visualization (Seaborn/Matplotlib), Streamlit, ETL.
